# YTRPC

🚀 A simple Discord Rich Presence client for Youtube Music.

## ⬇️ Installation

#### NPM

```bash
npm install @piarre/ytrpc
```

#### Docker

```bash
docker pull ghcr.io/piarre/ytrpc-server:latest
```
